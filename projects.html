Projects
"Wiki Dither"
React app that uses the Wikipedia API. Allows users to search Wikipedia pages and generate pixel art from page images using the Floyd-Steinberg dithering algorithm, giving a glimpse of what Wikipedia may have looked like in the 80s.
"Kallax"
Music library web app built on MongoDB, Node, Vue 3, and Express. uses the Discogs API to allow users to add and save music releases to a virtual library, view release metadata, create playlists, and find releases on YouTube.
"Teaspoon"
Recipe bookmarking web app built on React, Node, and Express. a custom API allows users to search for recipes on a number of cooking websites and compile them onto a single platform + user interface. also generates a grocery list from parsing recipe ingredients.
